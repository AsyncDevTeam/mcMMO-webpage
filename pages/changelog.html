v1.11.0 24 mars 2023

Support for German language

Goodbye mcMMO-Stats-Panel, welcome mcMMO-Dashboard !

New features

  • Add gorgeous loading bar
  • Improve top players visibility accross all pages (highlight their names)
  • Server version now supports color codes
  • Server icon is now updated in the header and in the tab
  • German translation thanks to @James_Bach
  • Hamburger menu icon is now a cross when open

Bug fixed

  • MDI icons were missing
  • Issue with wrong translations et column names

Config file

Please take the new parameters.js because structure was modified a lot. The german translation was added to parameters.js. New setting available in parameters.css to set horizontal size of the page (for large screens)

v1.10.0 20 mars 2023

Patching some errors

In order to fix some issues about wrong paths and unauthorized access on php scripts, we release this new version.

Bugs Fixed

  • Absolute not relative paths => to avoid that, all pages were moved to the root of the project
  • PHP scripts were not secured and denying all requests => to fix this, we implemented the CSRF token
v1.00.0 19 mars 2023

The beginning

We are proud to announce the launching of mcMMO Stats. After months of prototypes, we think this first version is the right one. Contribute to the project to help us improve mcMMO Stats.

Design

  • Customize mcMMO Stats appearance
  • Dark mode
  • Responsive design
  • Tips: Use your server logo to select the right colors

Features

  • View global server data directly in the homepage
  • Best server player
  • View user data for all your server
  • Search for a specific user
  • Visualize data in chart
